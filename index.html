System User Guide

Comprehensive instructions to navigate and utilize the Ghana School Terminal Report System

📖 Getting Started & Navigation

This application is designed specifically for schools following the Ghana Education Service (GES) Standards. It runs fully client-side on your browser, storing all data securely in your local browser database (localStorage). No data is sent to external servers, ensuring total privacy.

🏫 1. Setup Classes & Rosters

Navigate to the Classes tab. Click ＋ New Class to configure a class name, select academic year/term, choose subjects, and import or type names.

📝 2. Enter Student Marks

Go to the Enter Scores tab. Select your Class and Subject. Input CAT 1, CAT 2, Project, Group, and Exam marks. The system instantly scales SBA (50%) and Exams (50%).

📊 3. Check Rankings & Export

Under Class Results, view live class leaderboards, filter by grade levels, search names, and download reports as Excel-ready CSV files.

📄 4. Print Report Cards

Select a student in the Report Card tab. Fill in attendance and remarks. Click 🖨 Print Report Card to get an optimized print format.

📥 How to Batch Import Student Rosters

Avoid typing student names one-by-one by using the roster importer:

  1. Click New Class or Edit Class in the Classes view.
  2. Click the 📥 Import List button above the roster fields.
  3. Choose Copy & Paste to paste names directly from a spreadsheet column (e.g. Excel/Google Sheets), or select Upload CSV to upload a roster file.
  4. If the roster contains ID and Name separated by a comma (e.g. S005, John Doe), it maps them. If only names are pasted, the system generates IDs starting at S001.
  5. Click Apply Import to load names into the roster inputs, then click Save/Create Class.

💡 Tip: Re-importing into an existing class keeps previously entered student marks as long as the student ID remains unchanged.

📐 Ghana SBA Formula & Scaling Rules

The system complies with the new standardized 4-part SBA score entry and 50/50 scale:

  • SBA Component Limit (Max 60): CAT 1 (20) + CAT 2 (20) + Individual Project (10) + Group Project (10).
  • SBA Contribution (50%): SBA Total × (50 / 60).
  • Exam Contribution (50%): Exam score (Max 50).
  • Final Mark: Math.round(Scaled SBA + Exam Score).

💾 Database Backup, Restore, and Safety

Because all database entries reside on your current web browser, clearing cookies or resetting browser settings can erase your data. Protect your grades roster database by performing regular backups:

📦 Exporting Backups:

Go to the Branding & Sync tab, scroll down to the database section and click 💾 Export Backup (.json). Keep this file stored safely on your PC, USB drive, or cloud storage.

📥 Restoring Data:

If you change devices or clear your browser data, go back to Branding & Sync, click Restore Backup (.json), select your exported backup file, and your database will instantly recover.
